## 所感
* 最初のシナリオは小さくシンプルにしないと1周目が終わらない
* シナリオが決まりにくい
* DDDを読んだことあるのは1名
  * でもなんとかなった
  * 用語の説明等をあまりしないで進めてしまったのは申し訳なかったです
* 2周目終了時点でのモデリングは当たらずも遠からずな内容
* 実装して確認する時間が（案の定）足りなくなった
* プログラマ以外の方にも雰囲気をつかんで頂けた感じ
